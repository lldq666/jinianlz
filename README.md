# 纪念荔枝

这是一个纪念日正计时网页应用，用于记录和展示从2026年6月27日20:23开始至今的时间。

## 功能特点

- 实时正计时显示：精确到秒的时间计算
- 温馨浪漫的视觉设计：渐变背景、漂浮装饰元素
- 响应式布局：适配移动端和桌面端
- 精美的动画效果：淡入、滑动、漂浮等多种动画

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (状态管理)

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目已配置Vercel部署，推送到GitHub仓库后会自动部署。
